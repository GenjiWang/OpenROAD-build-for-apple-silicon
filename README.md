# Genjiwang Openroad

## How do I install these formulae?

`brew install genjiwang/openroad/<formula>`

Or `brew tap genjiwang/openroad` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "genjiwang/openroad"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
